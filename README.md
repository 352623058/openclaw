#  Agent-Driven Obsidian Vault

这是一个由本地大语言模型（OpenClaw）通过 `knowledge-pipeline` 技能全自动管理的 Obsidian 知识库。

## 自动化工作流声明
- **触发机制**：通过聊天发送 `Memo: ` 前缀，或通过每日晚间 21:00 定时整理触发。
- **Git 策略**：全自动化。脚本执行完毕后自动触发 `git add .` -> `commit` -> `push`。

## 核心目录架构与路由规则
本知识库遵循严格的读写权限隔离：

### 00-Inbox (默认收集箱)
- 存放：单次随口感慨、未定型想法。
- 规则：按天建表 `YYYY-MM-DD.md`，**仅限追加 (Append)**。

### 10-Projects (待办项目)
- 存放：强 Action Item（如“明天要”、“记得去”）。
- 规则：写入 `待办清单.md`，**仅限追加**。

### 20-Areas (核心领域)
- **日常流水**：`考研记录.md`、`股票投资.md`、`算法刷题.md`等（**仅限追加**）。
- **动态汇总**：`考研进度总览.md`、`考研计划表.md`（**Agent 拥有覆写 Overwrite 权限**，先读取合并再覆写）。
- **自动拓荒**：当出现全新的高价值专业体系时，Agent 会在此目录下自动创建新的 `.md` 文件并打上对应标签。
