## 奇偶性与反常积分

**核心原则：** 在利用"奇函数在对称区间积分为零"这一性质前，首要前提是该积分必须收敛。

**问题情景：**
- 积分区间包含无穷边界（如 $\int_{-\infty}^{+\infty} x\,dx$）
- 积分内部存在使函数无界的瑕点

**正确处理流程：**
1. 将区间拆开为独立的（广义）反常积分
2. 分别计算每一段的单侧极限
3. 若每一项都收敛 → 可继续用奇偶性
4. 若任意一项发散 → 整体变为 $\infty - \infty$ 未定式，原积分发散无解

**常见错误：**
- 直接认为左右对称抵消为 0（这实际是 Cauchy 主值，不等于原积分收敛）
- 对发散积分滥用奇偶性结论

**可微性判定流程（应试口诀）**

> 先看长相：是不是显然不连续的阶数不对等分式？
> 是 → 先证不连续 ⇒ 判死刑（不可微）⇒ 补算偏导数。
> 否 → 进入标准流程。

**标准流程**：
1. 算偏导
2. 拼可微定义式算极限
3. 如果可微 → 直接宣布连续
4. 如果不可微 → 再用极坐标算连续
